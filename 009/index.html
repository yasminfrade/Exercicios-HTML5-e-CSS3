Trabalhando com listas

Listas ordenadas

São listas em que a ordem faz a diferença

  1. Acordar
  2. Ligar pro João
  3. Tomar café
  4. Escovar os dentes
  5. Lavar roupa

Inserindo algo na lista antes de outro item, automaticamente todos os outros itens serão renumerados.
A tag <li> no HTML 5 não tem obrigatóriedade de ser fechada, é opcional.
Porém a tag <ol> é obrigatório o fechamento.
Dentro da tag <ol> existe os Types 1, A, a, I, i

Listas não ordemanadas

A ordem neste tipo de lista não importa.

  • Pão
  • Leite
  • Café
  • Requeijão

O item inserido fica exaatamente onde foi inserido.
As tags segue o mesmo principios, em que o fechamento de <li> nçao é opcional e <ul> é obrigatório.
Dentro da tag <ul> temos 3 opções de type: circle, square e disc
