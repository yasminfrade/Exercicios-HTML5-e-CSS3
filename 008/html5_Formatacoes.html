Principais formatações de texto

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semántica)

Nesta frase temos um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântico)

Nesta frase temos um termo em ênfase usando a tag <en> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

E no outro parágrafo temos outro texto marcado no final.

Texto Grande / Pequeno

Estamos criando um texto grande (tag <big>; não semântico) e outro texto pequeno (tag <small>; semântico) nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado (tag <del>).

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois (<ins>).

Existe também o texto sublinhado com a tag <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3
Usando a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O
Usando a tag <sub>
